# dotfiles-core

## Manual steps

For Win11 Terminal:
* Download and install https://github.com/ryanoasis/nerd-fonts/releases/latest/download/AdwaitaMono.zip
    - Mono = basic
    - Mono ... Mono = smaller icons in yazi
* Change tab keybindings
* Consider exporting `settings.json`


## Scripted steps

```
git clone https://github.com/khzs/dotfiles-core.git ~/.config/dotfiles-core
source ~/.config/dotfiles-core/one-time-setup-$(uname).sh 
```
